### 1. 什么是纯函数
不依赖系统的状态，不依赖且不改变作用域以外的变量，也就是说，如果参数相同，那么返回值也相同，输出完全取决于输入
纯函数是函数式变成的一个基础

```javascript
function sum(a, b) {
  return a + b
}
```

### 2. 优点
<!--more-->
> * 代码简洁，只关注输入输入
> * 函数目的很明确
> * 健壮
> * 容易测试
> * 更灵活
